## 기본형 vs 프리미엄 비교

| 항목 | 기본형 | 프리미엄 |
|------|--------|----------|
| 일일 호출수 | 5천회 | 10만회 |
| 시간당 최대 호출 | 제한 없음 | 1만회 |
| 목차/소개 정보 | ❌ 제공 안 됨 | ✅ 모두 제공 |
| 저자 상세 정보 | ❌ | ✅ |
| 판형 정보 | ❌ | ✅ |

## 프리미엄 전용 OptResult 파라미터

기본형에서는 요청해도 제공되지 않는 정보:

- `toc` - 목차
- `fullDescription` - 상품의 전체 설명 및 출판사 상품소개
- `authors` - 상품의 저자/아티스트 정보 목록
- `categoryIdList` - 전체 분야 정보
- `previewImgList` - 미리보기 이미지 경로
- `eventList` - 관련 이벤트 정보
- `Story` - 줄거리
- `mdrecommend` - 편집장의 선택
- `phraseList` - 책속에서 (최대 3개)

## 프리미엄 승인 조건

프리미엄 승인을 받기 위한 **필수 조건**:

### 1. 기본형 먼저 사용
- 기본형으로 이용 중인 상태에서 프리미엄 신청 가능
- 호출량 증대 또는 추가 DB 스펙이 필요한 경우

### 2. 출처 명시 (필수)
```
도서 DB 제공 : 알라딘 인터넷서점
```

### 3. 알라딘 링크 버튼 노출 (필수)
- 알라딘으로 연결되는 버튼이 노출되어야 함
- 책 정보 표시 시 알라딘 상품 페이지 링크 포함

### 4. 타 서점 링크 금지
- 알라딘 외 다른 서점 링크 제공 불가

### 5. 연 1회 정기점검
- 프리미엄 사용자는 연 1회 정기점검 대상

## 신청 방법

구글 폼을 통한 자료 제출:
1. 기본형 TTBKey 발급 후 서비스에 적용
2. 위의 조건을 충족하는 서비스 구현
3. 구글 폼으로 프리미엄 신청
